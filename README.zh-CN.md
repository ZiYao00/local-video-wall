# Local Civitai Video Wall

[English README](README.md)

一个在 Windows 本机浏览、筛选和整理 AI 生成视频与图片的本地媒体墙工具。

![预览](assets/preview01.gif)

它适合用来管理来自 Civitai、ComfyUI、Wan、Kling、Runway 等工具的本地 AI 媒体文件夹。工具会自动预览当前屏幕内可见的视频，滚出屏幕后暂停或释放资源；图片可以进入大图预览和全屏幻灯片；常用整理操作尽量保持本地、可见、可撤回。

下一阶段的开发方向是 AI 素材管理：读取 sidecar 元数据、显示提示词和模型信息、增加标签，并建立轻量本地索引。

## 功能特点

- 本地媒体文件夹扫描
- 支持图片 + 视频混合浏览
- 支持从图片弹窗进入全屏图片幻灯片
- 支持手动输入媒体目录，并提供单层文件夹路径自动补全
- 支持新版 Windows 文件夹选择框
- 支持记住路径
- 支持带内存缓存的左侧分栏侧边栏浏览磁盘、最近路径和收藏路径
- 支持扫描所选目录下最多 2 层子文件夹
- 可自行添加文件名排除关键词，并选择仅图片或图片和视频；默认关键词为 `fanart`、`thumb`
- 支持 2-20 列切换
- 2-9 列时自动按“列数 x 2”播放可见视频；10 列及以上切换为静态墙内预览
- 当前屏幕内视频自动静音循环播放
- 滚出屏幕后自动暂停，离开较远后释放资源；翻页、重扫、关闭弹窗时也会释放旧媒体资源
- 图片墙只加载接近可见区域的图片，远离后释放资源，不生成磁盘缩略图缓存
- 500MB 以上视频默认显示轻量占位封面，可在设置中允许墙内预览大文件
- 大型扫描结果支持输入每页数量，最高按 240 个卡片计算，避免一次创建上千个媒体元素
- 显示轻量性能统计，包括扫描耗时、渲染耗时、本页数量和已加载媒体数
- 扫描数量与审核状态显示在媒体网格上方右侧
- 支持搜索文件名
- 支持按文件大小和最近修改时间筛选
- 支持一键重置搜索和筛选，不影响当前路径、收藏或批量选择状态
- 支持排序：修改时间、文件名、文件大小
- 支持随机浏览
- 支持导出当前审核列表为 CSV
- 支持沉浸模式
- 支持暗色 / 亮色主题切换
- 支持小、标准、大三档界面字体
- 在宽屏下可将主网格和预览布局设置为居中、左侧或右侧
- 调整浏览器窗口宽度时，左侧文件夹分栏与媒体网格保持独立，不再互相覆盖
- 点击图片后可浮层大窗口预览，并支持上一张/下一张、鼠标滚轮切换、全屏幻灯片、默认应用打开和自动隐藏控制栏
- 点击视频后可浮层大窗口播放，并支持上一段/下一段、播放模式、滚轮音量、全屏、默认应用打开和自动隐藏控制栏
- 宽屏左侧 / 右侧布局下，图片和视频预览弹窗可跟随布局变成半屏预览
- 支持打开视频所在位置
- 支持安全移动到 `_video_wall_review` 或 Windows 系统回收站，并可选择是否确认
- 视频预览支持单视频循环、顺序播放和随机播放
- 幻灯片支持播放/暂停、上一张/下一张、1-15 秒间隔、效果、Fit、循环、全屏和自动隐藏控制栏
- 幻灯片效果支持无效果、淡入淡出、滑动、动态漂移和随机
- 默认英文界面，顶部工具栏可切换中文
- 顶部工具栏可直接筛选：全部、视频、图片、收藏
- 支持批量选择可见媒体，并进行收藏、取消收藏、移到回收站和导出 CSV

## 使用场景

这个工具适合：

- 浏览 Civitai 下载的视频和图片素材
- 浏览 AI 生成的视频和图片批次
- 浏览 ComfyUI / Stable Diffusion / Wan / Kling / Runway 等工具生成的本地输出
- 快速筛选大量本地媒体文件，不需要导入云端服务
- 像看动态素材墙一样浏览本地文件夹
- 为后续标签、元数据搜索、提示词和模型信息整理做准备

## 运行环境

- Windows 10 / Windows 11
- Python 3.10 或更高版本，推荐 Python 3.12
- 推荐使用 Chrome / Edge 浏览器

这个工具会以本地 Python 服务运行。如果电脑没有安装 Python，`start.bat` 和 `service.bat` 不能直接启动工具。

## 使用方法

### 1. 下载或克隆

普通用户建议：

1. 打开 GitHub Releases 页面。
2. 下载最新的 `local-civitai-video-wall-*.zip` 压缩包。
3. 解压到本地文件夹。
4. 运行 `start.bat`。

开发者也可以克隆仓库，并从源码运行。

### 2. 启动工具

手动模式：

双击：

```text
start.bat
```

浏览器会自动打开，并显示命令行窗口。关闭这个命令行窗口就等于退出工具。

服务模式：

```text
service.bat
```

这个文件会打开英文菜单，可以选择后台启动、停止或重启后台服务、安装开机自启、取消开机自启、打开浏览器、后台启动并打开浏览器或检查状态。开机启动会使用 Windows APPDATA 目录下的稳定辅助脚本，不再依赖临时目录。

也可以在命令行运行：

```bash
python app.py
```

然后访问：

```text
http://127.0.0.1:8787
```

### 3. 选择媒体目录

你可以：

- 手动输入视频文件夹路径，例如 `C:\Users\你的用户名\Videos`
- 点击侧边栏按钮，打开左侧分栏文件夹树
- 使用路径栏内部的星标按钮收藏当前路径
- 使用路径栏内部的历史按钮打开最近路径或清空历史记录
- 点击“Choose Folder / 选择文件夹”按钮
- 勾选“Remember path / 记住路径”
- 勾选“Scan subfolders / 扫描子文件夹”

从文件夹面板或 Windows 文件夹选择框选中路径后，会自动填入路径栏并开始扫描。手动输入路径时，可以按 Enter 或点击“Scan / 扫描”。

### 4. 浏览和整理图片 / 视频

- 使用顶部工具栏切换“全部 / 视频 / 图片 / 收藏”。
- 使用列数选择器调整媒体墙密度。2-9 列时，可见视频会按“列数 x 2”自动播放；10 列及以上使用静态墙内预览。
- 使用每页数量输入框控制每页渲染的卡片数量，最高 240。
- 点击图片可以打开图片预览窗口。
- 图片预览里可以使用左右按钮、键盘方向键或鼠标滚轮切换上一张/下一张。
- 点击“幻灯片（全屏）”可以进入图片幻灯片模式。
- 幻灯片模式下，可以在顶部工具栏控制播放/暂停、间隔、效果、Fit、循环和完整屏幕。
- 点击视频可以打开视频预览窗口。
- 视频预览里可以使用左右按钮切换上一段/下一段。
- 视频预览里可以使用鼠标滚轮调整音量。
- 视频播放模式支持单视频循环、顺序播放和随机播放。
- 图片、幻灯片和视频的观看控制栏会自动隐藏；需要操作时，把鼠标移到右上角控制区域即可重新显示。
- 点击批量按钮后，可以选择多个可见媒体，并执行收藏、取消收藏、移到回收站或导出 CSV。

### 5. 界面选项

顶部工具栏和设置菜单提供这些常用控制：

- 语言：英文 / 中文
- 主题：暗色 / 亮色
- 布局位置：居中 / 左侧 / 右侧
- 路径历史：可清空最近路径，收藏路径会保留
- 收藏路径和历史记录都支持单条移除
- 文件名排除规则：最多添加 30 个关键词，不区分大小写，可随时启用、停用或删除
- 大视频墙内预览：500MB 以上默认关闭，但点击占位封面仍可按需打开播放
- 界面字体：小 / 标准 / 大
- 网格列数：2-20 列
- 每页数量：1-240 个卡片
- 顶部筛选：全部 / 视频 / 图片 / 收藏

## 推荐设置

普通浏览和预览：

```text
6列
每页 120
```

看细节：

```text
4列 或 5列
每页 80-120
```

快速扫素材：

```text
8列 或 9列：适合自动播放预览
10-20列：适合静态总览
```

## 注意事项

浏览器自动播放视频通常要求视频静音，所以瀑布流里的视频默认是静音播放。500MB 以上的视频默认在墙内显示占位封面，避免误加载电影或演唱会等超大文件；可在设置中修改。

点击视频后，会打开大窗口播放器，你可以在播放器里手动开启声音。

## 配置文件

工具会在本地使用：

```text
config.json
```

保存你的设置，例如路径、列数、每页数量、排序方式、沉浸模式、语言、主题和字体大小。

幻灯片的间隔、效果、Fit 模式和循环设置也会保存在 `config.json`。

这个文件可能包含你的本地路径，不建议上传到 GitHub。上传仓库时保留 `config.example.json` 即可。

## 未来方向

当前版本可以作为稳定的本地媒体墙基础。后续计划会逐步转向 AI 素材管理，同时继续保持本地、轻量、Windows 优先。

建议顺序：

1. 稳定当前 `v1.7.x`：同步中英文文档、更新更新日志、补充基础测试清单。
2. 增加 `v1.7.5 - Architecture Guardrails`：先建立小范围后端模块边界，用于安全 JSON 读写、审核状态兼容和未来元数据标准化，但不改变当前运行行为。
3. 开发 `v1.8.0 - AI Metadata Preview`：读取 ComfyUI / Stable Diffusion WebUI 图片内嵌元数据、同名 sidecar JSON 和视频基础技术信息，并在媒体预览里显示提示词、负面提示词、模型、LoRA、来源和 Raw JSON。
4. 开发 `v1.9.0 - Tags & Review Workflow`：增加本地标签、评分、备注和批量打标签。
5. 开发 `v2.0.0 - Local Index`：增加轻量 SQLite 索引，用于加速重复扫描和元数据查询。
6. 开发 `v2.1.0 - Metadata Search`：按 Prompt、模型、LoRA、标签、来源 URL 和审核状态搜索筛选。
7. 视情况开发 `v2.2.0 - Optional Preview Cache`：在元数据和索引稳定后，再考虑可选图片缩略图和视频封面缓存。
8. 改进 `v2.3.0 - Packaging`：Release ZIP、启动检查、可选 portable tools 和更清晰的故障排查。

完整计划和第三方工具复用规则见 [ROADMAP.md](ROADMAP.md)。
